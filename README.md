- このサービスはどんなサービスか（3行以内）
こころのきろくは病気と向き合う人が服薬や日々の気持ちを記録し、同じ境遇の人たちと緩やかにつながれるサービスです。
記録を続けることで自分を振り返り、利用者同士のいいねを通じて励ましあえる、闘病中の不安感を少しでも和らげることを目的としています


- 誰の、どんな課題を、どう解決するか
継続的な通院や服薬が必要な方が、服薬記録や薬の副作用、体調の変化などを記録し、治療や自己管理に役立てることができます
また、闘病中に感じる不安感や孤独感を、同じように病気と向き合う利用者同士が励まし合うことで軽減することを目指します。

## 2. このアイデアはどこから生まれたか

自身の闘病生活の中

### 2-1. きっかけとなった体験・感情

わたしは精神疾患を抱えており、現在も通院や服薬を続けています
服薬を続ける中で、通院時に振り返れる日記のようなアプリがあればいいなと思いました
しかし日々の記録をつけるものはあれど、SNSのような機能があるものはあまり見当たらず、SNSほどではない、SNSほどは疲れないような、日々の記録を励ましあえるアプリがあればいいなと思いました

### 2-2. なぜそれが「気になった」のか

病気そのもののつらさだけではなく、孤独感や不安感も大きな負担になると感じています
そのため自身の記録だけでなく、誰かからの共感や応援を感じられるとより良いものになるのではと思いました
